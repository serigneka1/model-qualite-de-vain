PROJET: Modèle de machine learning pour prédire le score de qualité du vin

PROBLEMATIQUE:
_ Production de vin, un marché lucratif
- Une erreur peut coûter beaucoup d'argent surtout dans le cas où le vin n'est pas certifé de bonne qualité

Comment à la fois améliorer la qualité du vin dans le cadre d'une certification en utilisant une approche scientifique et aider les producteurs à anticiper la qualité du vin pour se positionner les opportunités du marché et augmenter leur CA ? 

**La certification permet à la fois de protèger le vin contre toute falsification et aussi assure sa qualité.**

**L'objectif c'est alors de terminer les paramètres qui influent le plus sur la qualité du vin.**
